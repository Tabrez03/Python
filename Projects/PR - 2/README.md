# Pattern Generator and Number Analyzer

A beginner-friendly Python project that demonstrates the use of loops, conditional statements, user input, and menu-driven programming.

## Features

### 1. Pattern Generator

* Generates a right-angled triangle star (`*`) pattern.
* User can choose the number of rows.

**Example:**

```text
*
* *
* * *
* * * *
```

### 2. Number Analyzer

* Analyzes a range of numbers provided by the user.
* Identifies whether each number is **even** or **odd**.
* Calculates and displays the **sum** of all numbers in the range.

### 3. Exit Option

* Allows users to safely exit the program.

## Technologies Used

* Python 3.10+
* Match-Case Statements
* Loops (`for`, `while`)
* Conditional Statements (`if-else`)
* User Input Handling

## How to Run

1. Make sure Python is installed on your computer.
2. Clone this repository:

```bash
git clone https://github.com/your-username/pattern-generator-number-analyzer.git
```

3. Navigate to the project folder:

```bash
cd pattern-generator-number-analyzer
```

4. Run the program:

```bash
python main.py
```

## Program Menu

2. Analyze a Range of Numbers
