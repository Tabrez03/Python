# Interactive Personal Data Collector 📝

A beginner-friendly Python program that collects personal information from the user and displays the data along with its type and memory address.

## 🚀 Features

* Collects user details:

  * Name
  * Age
  * Height
  * Favorite Number
* Displays the data type of each input.
* Shows the memory address of stored variables using `id()`.
* Calculates the approximate birth year based on the user's age.

## 📋 Requirements

* Python 3.x

## ▶️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/interactive-personal-data-collector.git
   ```

2. Navigate to the project folder:

   ```bash
   cd interactive-personal-data-collector
   ```

3. Run the program:

   ```bash
   python main.py
   ```
